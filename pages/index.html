Welcome to DeckCreator!

F.A.Q.

What is DeckCreator?

An app to create decks for the MRBC mod in the Tabletop Simulator.

I don't have Tabletop Simulator, can I still create a deck?

Yes, you can. Using saved decks on Tabletop Simulator is just a plus.

What is Tabletop Simulator?

A great game that allows people to play Board Games online and creating mods of custom games.

Check it out TTS on steam!

Can I change the path where I save my decks?

Currently, no. For now it's just to be easy to use with TTS.

Does DeckCreator works on Linux?

Yes, it does! Although, it does not autoupdates in Linux/OS.

Does DeckCreator works on Mac?

Currently, no.

Top Menu:

  • Clear Cache
  • Sync Decks
  • Save Deck (Editor Page)

 

Tabletop Simulator usually saves the mods image files to the computer of all players entering the server. It does that to maintain a local cache and preload the images, in our case, all the card images so you do not have to download them every time it enters the server. The problem this generates when there is any change in the mod whatsoever is that your local images will not match the server images. That is, what you see is not the same as what appears to other people.

So click this option before entering the server when there is a change in the mod to force Tabletop download the mod again. Or you can also use this option to simply clear the folder and remove old TTS files that are no longer used to free up space on your computer.

Windows default path: %USERPROFILE%\Documents\Mods\Images

Linux default path ~/.local/share/Tabletop Simulator/Mods/Images

 

If you have switched computers and have decks saved in the cloud that are not yet downloaded to your new computer, use this option to download all and sync to your local folder.

Windows default path: %USERPROFILE%\Documents\My Games\Tabletop Simulator\Saves\Saved Objects\DeckCreator

Linux default path ~/.local/share/Tabletop Simulator/Saves/Saved Objects/DeckCreator

 

The app will save your deck in the cloud and in the .json format directly to the DeckCreator folder within the Tabletop Simulator directory so that it can be easily imported into the game.

Windows default path: %USERPROFILE%\Documents\My Games\Tabletop Simulator\Saves\Saved Objects\DeckCreator

Linux default path ~/.local/share/Tabletop Simulator/Saves/Saved Objects/DeckCreator
